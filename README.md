# Tauri 2 Svelte 5 Typesafe Boilerplate

> [!NOTE]  
> Some libraries are currently in alpha version.

## Setup
```bash
git clone https://github.com/lanius412/tauri2-svelte5-typesafe-boilerplate.git
cd tauri2-svelte5-typesafe-boilerplate
bun i
```

## Development
```bash
bun tauri dev
```
